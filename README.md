Controls

Move: Arrow Keys
Attack: E
Jump: Spacebar
Block: Q
Dash: Left Control

Debug Controls:
Teleport to next Checkpoint: K
Show / Hide Game Stats: 2 / 3
God Mode On / Off: 9/0
Back to Checkpoint / Delete Save Data: O / P
